Final Year Hons Degree Dissertation project in IT & IS, DIT Kevin St.

Speciality Coffee listings and route finding and locator for Dublin. Web delivered.

Synopsis

Coffee

Coffee, its all about the coffee afterall. Search by roaster, I don't go as far as region or blend, perhaps if somebody pays me to do a phase 2, I might :).

Shops

Theres no point in me giving you all this information about the coffee on offer if you cant find out where to buy it from.

Map

I've cooked up a custom set of tiles using tilemill studio, so I can control what appears on my maps. I can also customise the details which I think are more important to the users.

Details

No point in my giving directions and listings of shops without details about the shops themselves, click the maplink on the menubar to find details about each including:

  • wifi
  • power points
  • table service
  • seating

Routing

I've used PostGres as the database and implemented the PostGIS addon for geospatial queries, I've also implemented PG-Routing to enable me to give route information from your current location to a shop of your choosing
